Build your first Web App for Data Visualization

2020.04.24

Build your first Web App for Data Visualization using Vue.js!

In this article we’re using two modern tools for Data Visualization using Javascript:

  • Vue.js - about vuejs
  • eCharts - about echarts

Assumptions:

  • Minimal knowledge of Javascript;
  • We are using free and open-source software package manager to simplify the installation of software. Here we’ll use Homebrew (macOS >= 10.14 or Linux). You can use other package manager for your OS or install Node and Yarn following their instructions.

1. Javascript (Node + Package Manager) Setup:

  • Cleanup previously installed packages. Using Homebrew:

    brew uninstall --force yarn node npm
brew cleanup
brew update

  • Install Node. Using Homebrew:

    brew install node

Yarn and NPM are both Node package managers - I particularly prefer Yarn.

  • Install Yarn. Using Homebrew: 
    npm install -g yarn

2. Vue.js Setup

We’re using Vue CLI to start an application quickly (the CLI assumes prior knowledge of Node.js and the associated build tools)

  • Install Vue CLI: 
    yarn global add @vue/cli

Check Vue.js website for more info

3. Application Setup

  • Create an application called “web-visualization” using this command:

    vue create data-visualization
cd data-visualization
yarn serve

  • Installing the chart packages (eCharts and vue-echarts, a wrapper for eCharts)

yarn add echarts vue-echarts

yarn add -D pug pug-plain-loader
yarn add -D stylus stylus-loader
