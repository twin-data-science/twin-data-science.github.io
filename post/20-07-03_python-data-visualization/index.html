2020.07.03
development data-science data-visualization python

Premissas para este post: Dê uma olhada neste post * sobre ferramentas para começar com Python

A Visualização de Dados é a apresentação gráfica e um contexto visual para os padrões, informações e correlações que analisamos. Existem excelentes pacotes gráficos para Python (Matplotlib, Seaborn, Pandas Visualization …)

Podemos mostrar o propósito da visualização de dados com o exemplo:

  • mostrar conjunto de dados cru
  • mostrar gráfico plotado

Neste post, vamos começar utilizando o Matplotlib que é uma das ferramentas de visualização de dados mais utilizadas por cientistas de dados. Futuramente, falaremos de forma interativas de plotar gráficos, como acontece com o Plotly, que é baseado em D3 e também pode ser utilizado em JavaScript.

Também utilizaremos o pandas, a biblioteca mais popular em Python para análise de dados, para auxiliar na leitura e manejo dos dados.

1. Instale as bibliotecas:

Abra seu terminal e instale as bibliotecas que precisaremos utilizar:

$ pip3 install matplotlib
$ pip3 install pandas
2. Crie um novo diretório:

Ainda pelo seu terminal, crie um novo projeto e acesse-o (como já recomendado*, crie e acesse um diretório central para desenvolvimento):

$ mkdir python-datavis
$ cd python-datavis
3. Inicialize o Jupyter Lab

Em seguida, inicialize o Jupyter com o comando:

$ jupyter lab

Na primeira célula do seu Notebook, importe a biblioteca Matplotlib import matplotlib.pyplot as plt . Logo abaixo da importação, digite o comando %matplotlib inline, que permite a plotagem de imagens estáticas incorporadas no seu Notebook
