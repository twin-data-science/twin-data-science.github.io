Ajeitando sua máquina - Python

July 2020
Development / Python

$$ \sum_{i=1}^{N} a_i $$

Python é uma das linguagens de programação favoritas para ciência de dados, aprendizado de máquina e processamento de linguagem natural. A linguagem é muito popular por sua fácil sintaxe, ela é simples de aprender e utilizar. Suas bibliotecas, módulos e frameworks são amplamente conhecidos, apoiados e mantidos por corporações renomadas. É uma linguagem versátil, que pode ser utilizada em praticamente qualquer ambiente, para a solução e automação de diferentes tarefas.

Por estes motivos e vários outros, a maioria dos posts sobre ciência de dados deste blog serão em Python. Para executá-los, você vai precisar de algumas configurações e ferramentas importantes instaladas e funcionando na sua máquina para começarmos a desenvolver. A seguir estão descritos os primeiros passos:

Utilize uma interface para Linha de Comando

Para desenvolver em Python (ou qualquer outra linguagem), você vai precisar acessar sua interface com a linha comando da sua máquina. Caso utilize macOS / linux, abra a aplicação Terminal. Se você é usuário de windows, abra o prompt de comando (‘Command Prompt’).

Obs: Para padronizar, chamarei sempre de Terminal. Você também poderá encontrar escrito como: Interface de Linha de Comando; Command Line Interface (CLI); Console; Prompt de Comando.

Crie um diretório de desenvolvimento

Uma boa dica é manter seus diretórios de desenvolvimento isolados das demais no seu computador. Gosto de criar uma pasta chamada “Developer” no diretório raíz do meu computador, para que os códigos fiquem em um local centralizado. Para fazer isso no macOS / linux / Windows, abra seu terminal e execute o comando para localizar o diretório raíz:

$ cd ~/

A caixinha acima significa que estou demonstrando um comando ou trecho de código. Caso comece o símbolo ‘$’, significa que estou descrevendo um comando no terminal (caso for copiar e colar para o seu terminal, ignore o ‘$ ' e copie apenas o resto).

Em seguida execute:

$ mkdir Developer

Este comando vai criar um novo diretório chamada Developer (geralmente lado a lado de outras pastas como “Desktop” / “Documents” / “Pictures” / “Music” / etc.). Para acessar esta pasta pelo terminal, a partir do diretório raíz e execute:

$ cd Developer

desta forma, o diretório de trabalho será agora ‘Developer’. Lá dentro você pode criar mais sub-pastas para novos projetos (exemplo: $ mkdir new-project) e acessar estes novos diretórios (exemplo: $ cd new-project).

Obs: Para padronizar, tentarei chamar de diretório (conhecido popularmente como ‘pasta’).

Instale o Python

Execute o seguinte comando para verificar se você já tem Python 3 instalado na sua máquina:

$ python3 --version

Se não estiver instalado, utilize as instruções retornadas pelo comando acima ou siga as instruções do site oficial. Use o Python 3.x, que é a versão com suporte e em desenvolvimento ativos (por mais que o Python 2.x possa vir na sua máquina).

Para simplificar esta e outras instalações, você também pode utilizar um gerenciador de pacotes, como o nativo do linux (apt ou similar) ou o Homebrew (também disponível no macOS). Após instalação, você instalará facilmente o Python 3 executando o comando: $ brew install python

Executar apenas o comando $ python geralmente inicializa o interpretador de Python 2. Executar o comando $ python3 , inicializará o interpretador correto. Digite ‘exit()’ (ou utilize o atalho Ctrl + Z no Windows | Ctrl + D no macOS / linux) para finalizar a execução do interpretador.

Os pacotes (e bibliotecas) Python, são um conjunto de funções úteis “empacotados”, sendo que cada pacote terá sua finalidade particular. Ao importar um pacote, você terá acesso às respectivas funções que alguém já implementou por nós. Por exemplo, posso utilizar um pacote de visualização (como o Matplotlib) para não ter que reescrever do zero todas as funções de plotagem de gráficos, conseguindo assim apresentar um gráfico de pizza chamando uma simples função.

A maioria das distribuições do Python vem com o gerenciador de pacotes pip pré-instalado, você pode checar sua existência executando o comando $ pip3 --version. Caso não esteja instalado, instale o pip para gerenciar suas bibliotecas Python.

Assim como o comando $ python, apenas $ pip instala pacotes Python 2, portanto utilize $ pip3 para o Python 3. O pip torna a instalação de pacotes Python muito mais simples e organizada. Você já pode testá-lo no próximo tópico, instalando o Jupyter.

Jupyter Notebook / JupyterLab

Outra ferramenta muito utilizada para Ciência de Dados é o Jupyter Notebook. Esta ferramenta viabiliza a combinação de código executável em tempo real, textos descritivos (Markdown) e outras visualizações interativas, em um documento com a extensão “*.ipynb”. Instale o Jupyter com o comando:

$ pip3 install jupyter

JupyterLab é a interface que aprimora a experiência dos notebooks, com uma estrutura modular, permitindo abrir vários tipos de arquivos em diferentes guias na mesma janela. Oferecendo uma experiência semelhante ao IDE. Você pode instalar o JupyterLab utilizando o comando:

$ pip3 install jupyterlab

Para inicializar o Jupyter Lab, é recomendável sua inicialização dentro do seu devido projeto. Por exemplo, se estamos desenvolvendo em um diretório chamado ‘my-project’, na sua pasta Developer, você deve acessar o diretório pelo terminal com o comando $ cd Developer/my-project e em seguida executar $ jupyter lab no seu console.

Para utilizar os Notebooks, acesse os links apresentados no console, geralmente acessando o endereço “http://localhost:8888/” em seu navegador. Deixe seu terminal aberto e executando sempre que estiver usando o Jupyter. Caso precise do terminal, abra uma nova aba ou janela do terminal em paralelo, interromper o terminal interromperá seu acesso ao Notebook e você pode perder alterações não salvas.

Para criar um novo documento, clique em “Notebook - Python 3”: O Jupyter abrirá uma nova aba com um novo arquivo chamado “Untitled.ipynb” A partir deste Notebook você pode começar a programar. Você também pode gerar documentação em Markdown (texto sem programação) em células, alterando o tipo da célula. Você pode renomear seu Notebook e salvar seu progresso (símbolo de disquete na aba aberta). Para encerrar sua execução, salve seu progresso e utilize o atalho Ctrl + Z no Windows ou Ctrl + D no macOS / linux.

Escolha um bom Editor de Texto

Gosto de usar VSCodium, a versão Free/Libre Open Source Software do editor de texto da Microsoft, VSCode (o código fonte do vscode contém telemetria/rastreamento). Se não se importar, você também pode usar o VSCode ou qualquer outro editor de texto de sua preferência (Atom, Notepad++, etc.) Se você tiver o Homebrew instalado, execute:

$ brew cask install vscodium

Também é possível ter uma experiência semelhante ao Jupyer no VSCodium / VSCode, já que há suporte para arquivos “*.ipynb” e que gera uma visualização de Notebook no editor de texto.

Um pouco mais avaçado

Se você já se sente confortável utilizando o terminal no macOS / linux, recomendo dar um upgrade no visual e usabilidade dele com o Oh My Zsh, utilize a instruções deste link.

Também é possível criar um alias para python3 (alias python="python3") e pip3 (alias pip="pip3"). Desta forma você não vai precisar digitar o ‘3’ ao final de cada um dos comandos.
